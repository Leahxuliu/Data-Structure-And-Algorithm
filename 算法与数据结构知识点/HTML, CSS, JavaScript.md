## 3.4 list

* `<ul>`标记来创建无序列表（如项目符号点）
  * unorder list
* `<ol>`带数字的有序列表
  * order list

```html
<!DOCTYPE html>

<!-- Demonstrates (unordered) lists -->

<html lang="en">
    <head>
        <title>list</title>
    </head>
    <body>
        <ul>
            <li>foo</li>
            <li>bar</li>
            <li>baz</li>
        </ul>
    </body>
</html>
```



## 3.5 table

* Tables start with a `<table>` tag and have `<tr>` tags as rows, and `<td>` tags for individual cells
  * tr: table row

```html
<!DOCTYPE html>

<!-- Demonstrates table -->

<html lang="en">
    <head>
        <title>table</title>
    </head>
    <body>
        <table>
            <tr>
                <td>1</td>
                <td>2</td>
                <td>3</td>
            </tr>
            <tr>
                <td>4</td>
                <td>5</td>
                <td>6</td>
            </tr>
            <tr>
                <td>7</td>
                <td>8</td>
                <td>9</td>
            </tr>
            <tr>
                <td>*</td>
                <td>0</td>
                <td>#</td>
            </tr>
        </table>
    </body>
</html>

```

## 3.7 link

* 添加link
* <a>是所有的设置，要分开设置的话，加上id
* link的字体是红色，且没有下划线
* 当光标靠近的时候，有下划线

```html
<!DOCTYPE html>

<!-- Demonstrates link -->

<html lang="en">
    <head>
        <style>
            #Harvard
            {
                color: #ff0000;
            }

            #CS50
            {
                color: #0000ff;
            }

            a
            {
                text-decoration: none;
            }

            a: hover
            {
                text-decoration: underline;
            }

        </style>
        <title>link</title>
    </head>
    <body>
       Visit <a href="image.html" id="Harvard">Harvard</a>.
       Visit <a href="https://cs50.harvard.edu/x/2021/notes/8/" id = "Yale">CS50</a>.
    </body>
</html>
```
